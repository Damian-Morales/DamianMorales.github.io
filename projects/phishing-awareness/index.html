Overview

This project generates preview-only phishing emails, routes users to a debrief landing page when they click, and records actions in an admin dashboard. It’s designed for awareness and training, not malicious use, and provided end-to-end practice with Flask, templating, and small database operations for campaign tracking.

Environment & Setup

Create virtual env & install

python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt

Initialize the database

python manage.py init-db

Configure environment

Copy .env.example to .env and adjust defaults.

Run a Campaign (Preview Mode)

python manage.py send --csv recipients.csv \
  --template emails/fake_invoice.html --campaign "October Drill"
