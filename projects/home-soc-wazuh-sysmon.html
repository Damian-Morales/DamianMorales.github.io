Home SOC: Wazuh SIEM + Sysmon Telemetry

This project demonstrates how to build a home-based Security Operations Center (SOC) using the open-source Wazuh SIEM and Windows Sysmon to collect and analyze event data.

Overview

I deployed Wazuh in a Dockerized Ubuntu environment and connected a Windows endpoint configured with Sysmon for rich telemetry. This setup enables detection, analysis, and incident response workflows similar to enterprise SOC environments.
