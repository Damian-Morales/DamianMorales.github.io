Iterating with Loops in Python Notebooks – My Lab

Overview

In this lab, I practiced writing iterative statements in Python to automate repetitive security tasks. I learned how to use for loops to process lists of items a set number of times, and while loops to continue until a condition is met.

Scenario

The notebook guided me through three exercises:

  • Displaying connection attempt messages for a list of IP addresses with a for loop.
  • Detecting which IPs in that list tried to reach restricted data and flagging them.
  • Generating a series of employee ID numbers in a while loop to simulate user account creation.

I implemented both loop types, incorporated comparison operators like <, <=, and ==, and verified the output for each iteration.

Conclusion

I now understand how to apply for loops for fixed iterations and while loops for conditional repetition. These constructs will let me automate routine security processes—saving time and reducing manual effort in future analyses.
