Course 3: Connect and Protect – Networks and Network Security

Course Overview

In this course, I explored network design, communication protocols, network threats, and how to secure systems against intrusion. I also learned about firewalls, hardening techniques, and strategies to protect both local and cloud-based infrastructure.

Module 1: Network Architecture

I learned how different types of networks are structured and how data travels through them. This included the TCP/IP model, physical components of a network, and foundational architecture concepts.

  • Types of networks and network architecture
  • TCP/IP communication framework
  • How data moves through a network
  • Physical network components

Module 2: Network Operations

This module covered communication protocols, wireless access methods, and tools used to keep network operations secure and reliable. I explored the role of firewalls and security protocols.

  • Network and wireless protocols
  • Firewalls and their functions
  • Security tools that protect network operations

Module 3: Secure Against Network Intrusions

I studied various attack methods used to compromise networks and the tactics analysts use to detect, respond to, and investigate intrusions.

  • Types of network attacks and breaches
  • How to secure networks from intrusion
  • Security investigation tools and tactics
  • Basic network troubleshooting

Module 4: Security Hardening

In this final module, I learned how to proactively defend systems through OS and network hardening. I also studied cloud security practices to protect modern infrastructure.

  • OS, network, and cloud hardening techniques
  • Preventive security strategies
  • Cloud-based security controls
