Developing Algorithms in Python Notebooks – My Lab

Overview

In this lab, I wrote a Python algorithm to verify that a user is authorized on the system and that the device they bring matches the assigned asset list. I practiced list indexing, element pairing, and using list methods within a function to automate decision logic.

Scenario

The notebook guided me through building a function that:

  • Accepts two lists: one of approved usernames and one of their corresponding device IDs.
  • Checks if a given username exists in the approved list (using .index() to locate it).
  • Uses the same index to verify that the provided device ID matches the assigned one.
  • Returns a message indicating “Access granted” or “Access denied” based on these checks.

Conclusion

  • List indexing (my_list[index]) works the same as string indexing and starts at 0.
  • .append() adds new elements; .remove() deletes them.
  • .index() locates items in lists (and strings) by returning their position.
  • Pairing two lists via shared indices lets me associate related data (users ↔ devices).
  • Encapsulating this logic in a function creates a reusable algorithm for authorization checks.
