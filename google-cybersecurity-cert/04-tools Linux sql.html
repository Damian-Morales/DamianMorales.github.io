Course 4: Tools of the Trade – Linux and SQL

Course Overview

This course introduced the tools used to manage operating systems and databases. I explored how cybersecurity professionals use Linux, the command line, and SQL to investigate security incidents and maintain system integrity.

Module 1: Introduction to Operating Systems

I learned about the purpose of operating systems and how they manage hardware, applications, and user interaction. I explored both graphical and command-line interfaces.

  • Main functions of an operating system
  • Common OS types and their use
  • OS interaction with hardware and apps
  • GUI vs. command-line interfaces

Module 2: The Linux Operating System

This module introduced me to Linux architecture and its importance in cybersecurity. I reviewed Linux distributions and learned about the shell interface.

  • Linux's role in security
  • Linux system architecture
  • Unique features of Linux distributions
  • The shell as a text-based interface

Module 3: Linux Commands in the Bash Shell

I practiced using Bash commands to navigate, manage, and secure Linux systems. This included managing file permissions and running administrative commands with sudo.

  • File system navigation with Bash
  • Managing files and permissions
  • Authentication and authorization
  • Using sudo for root access

Module 4: Databases and SQL

I learned to use SQL to query and manipulate data in relational databases. I used joins and filters to identify patterns and support investigations.

  • SQL’s use in cybersecurity
  • Relational database structure
  • Filtering data with SQL queries
  • Joining tables for combined insights
