Course 1: Foundations of Cybersecurity

Course Overview

This course introduced the foundational concepts of cybersecurity, including core skills, career paths, types of threats, risk management, and the tools used by entry-level analysts. I explored the importance of cybersecurity in modern organizations and the evolving role of a security analyst.

Module 1: Welcome to the Exciting World of Cybersecurity

I was introduced to the certificate program, its structure, and the field of cybersecurity. The module defined the role of a security analyst and emphasized key skills needed to succeed in this career.

  • How this program prepares learners for a security career
  • Definition of security and analyst job roles
  • Core skills and responsibilities of an entry-level analyst
  • How analysts protect networks and data

Module 2: The Evolution of Cybersecurity

This module covered the development of cybersecurity in response to growing digital threats. I learned how past and present attacks shaped the field and gained an introduction to the eight CISSP security domains.

  • Common types of attacks (past and present)
  • Impact of security breaches on businesses
  • How attacks shaped modern cybersecurity practices
  • Overview of the eight CISSP security domains

Module 3: Protect Against Threats, Risks, and Vulnerabilities

I explored the CIA triad, the NIST Cybersecurity Framework (CSF), and how organizations use these models to define risk management strategies. The module also introduced cybersecurity ethics.

  • Security frameworks and controls
  • CIA triad and NIST CSF
  • Using frameworks to reduce threats and vulnerabilities
  • Ethical responsibilities in cybersecurity

Module 4: Cybersecurity Tools and Programming Languages

I was introduced to the tools and languages security analysts use to detect, analyze, and respond to threats. This included SIEM tools, protocol analyzers, and programming with Python and SQL.

  • Common tools used by security analysts
  • Purpose and function of those tools
  • Basic use of programming languages like SQL and Python
  • How analysts use tools/languages to reduce risk
