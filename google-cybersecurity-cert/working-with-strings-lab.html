Working with Strings in Python Notebooks – My Lab

Overview

In this lab, I practiced manipulating string data in Python. I created and updated employee IDs and device IDs, extracted substrings from those values, and parsed components from URLs using slicing and built-in string methods.

Scenario

The notebook simulated tasks for a security analyst:

  • Updating an employee’s ID number by concatenating a department code and numeric suffix.
  • Extracting specific characters from a device ID to determine its model and region.
  • Parsing a URL string to isolate protocol, domain, and path components.

I applied `str()`, `len()`, and `.index()` to convert data to strings, measure length, and locate substrings as part of these tasks.

Conclusion

  • Strings store critical security data like IDs and URLs.
  • Concatenation (`+`) merges multiple strings into one.
  • Slicing allows extracting subsections: e.g. `device_id[3:6]`.
  • Built-in functions (`type()`, `str()`, `len()`) and methods (`.index()`) help convert, measure, and locate data within strings.
