Working with Built-In Functions in Python Notebooks – My Lab

Overview

In this lab, I explored Python’s built-in functions to work with lists of failed login attempts. I used sorted() to order monthly data, max() to find the peak value, and print() to display results. I then defined a custom function with a return statement to compare today’s attempts against the average.

Scenario

The notebook provided a list of monthly failed login counts. My tasks were to:

  • Sort the list in ascending order using sorted().
  • Identify the maximum number of failures with max().
  • Write a function that accepts today’s count, computes the average of the list, and returns a message indicating whether today’s attempts exceed, match, or fall below that average.

Conclusion

  • Functions can display output or return values to be stored in variables.
  • sorted() returns a new list sorted in numerical order.
  • max() returns the largest element in an iterable.
  • print() outputs information directly to the notebook.
  • Including a return statement lets a function pass data back for further use.
