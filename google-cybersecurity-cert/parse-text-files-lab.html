Parsing Security Logs from Text Files – My Lab

Overview

In this lab, I used Python’s file I/O syntax to import a raw log file, read its contents, and split the data into usable pieces. I also created a secondary text file to record a list of approved IP addresses for later analysis.

Scenario

As a security analyst, I needed to prepare my log data for deeper investigation. I wrote Python code to:

  • Open data/login.txt in read mode using the with statement.
  • Use .read() to load the entire log into memory, then .split() to break it into individual entries.
  • Create or append to allowed_ips.txt in write mode, writing each verified IP address on its own line with .write().

This approach ensured efficient and safe handling of file resources.

Conclusion

  • The with statement automatically handles opening and closing files.
  • The open() function takes:
    • "r" for reading,
    • "a" for appending,
    • "w" for writing.
  • .read() loads file contents as a single string; .split() converts it into a list.
  • .write() lets you append or overwrite text in a file.
