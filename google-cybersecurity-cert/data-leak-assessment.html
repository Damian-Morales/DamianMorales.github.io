Data Leak Assessment

Course:

Google Cybersecurity Certificate - Course 5

Lab Type:

Data Risk Assessment & Recommendation

Overview

In this lab, I evaluated a data risk scenario involving a leak caused by improper access controls. I reviewed the incident, studied the implemented controls, and used the NIST SP 800-53: AC-6 guidelines to recommend improvements to data privacy policies. The goal was to determine whether effective data handling processes are being used and how to enhance them using security frameworks like the NIST Cybersecurity Framework (CSF).

Steps Completed

  • Reviewed the incident summary involving a sales team accidentally sharing confidential documents via a public link.
  • Evaluated how least privilege controls were not enforced, contributing to the incident.
  • Consulted NIST SP 800-53: AC-6 to analyze current access control effectiveness.
  • Recommended two control enhancements:
    • Restrict access to sensitive resources based on user roles.
    • Regularly audit user privileges.
  • Justified how these recommendations will reduce future data leaks by limiting unnecessary access and improving oversight.
