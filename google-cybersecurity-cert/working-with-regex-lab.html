Working with Regular Expressions in Python Notebooks – My Lab

Overview

In this lab, I used Python’s re module to build patterns that automatically extract device IDs and IP addresses from log text. I practiced crafting regex sequences and applying functions like re.findall() to scan strings for matches.

Scenario

As a security analyst, I needed to:

  • Identify device IDs matching an OS-specific format (e.g., alphanumeric sequences) from a sample log.
  • Extract all IPv4 addresses present, then compare them against a list of flagged IPs for further review.

I defined patterns using tokens such as \w, \d, quantifiers (+{n,m}), and escape characters (e.g., \.), and used re.findall() to collect the results.

Conclusion

  • Regular expressions let me create flexible patterns to locate key data in text.
  • Regex symbols I used:
    • \w – any alphanumeric character
    • + – one or more occurrences
    • \d – any digit
    • \. – literal period
    • {x,y} – between x and y repeats
  • re.findall() returns all matches of a pattern in a string as a list.
  • Using regex in Python automates log parsing tasks like identifying vulnerable devices or suspicious IPs.
