Defining and Calling Functions in Python Notebooks – My Lab

Overview

In this lab, I practiced creating my own Python functions within a notebook. I learned how to write a function header using def, include an indented body of instructions, and invoke the function by name to automate repeated steps.

Scenario

The notebook guided me through two tasks:

  • Defining and calling a function that displays a security alert message when potential issues arise.
  • Writing a function to concatenate a list of employee usernames into a single string for reporting.

I combined parameters, return values, and string concatenation with the + operator to ensure the output matched the lab’s requirements.

Conclusion

Key takeaways from this lab:

  • I can define functions with the def keyword, a name, parentheses, and a colon.
  • The indented block under the header forms the function body, which runs when the function is called.
  • I used string concatenation inside a function to merge list items into one string using the + operator.
